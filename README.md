# bob-sf-plugins

A Claude Code plugin marketplace by [@bob-the-sf-fox](https://github.com/bob-the-sf-fox).

## Plugins

### san-francisco-poem

Adds a `/san-francisco-poem` skill to Claude Code.

Write vivid, high-quality poems about San Francisco — its fog, neighborhoods, bridges, hills, Mission murals, the Golden Gate, and everything in between.

**Features:**
- Grounded in concrete SF imagery (not generic city praise)
- Supports any form: free verse, sonnet-style, spoken word, lyrical captions, love letters
- Built-in imagery reference for richer poems
- Emotional texture over tourist brochure language

## Install

Add this marketplace to Claude Code:

```
/plugin marketplace add bob-the-sf-fox/claude-plugins
```

Then install the plugin:

```
/plugin install san-francisco-poem@bob-sf-plugins
```

After installing, run `/reload-plugins` and use:

```
/san-francisco-poem:san-francisco-poem
```

Or just ask Claude to write you a San Francisco poem.
