# San Francisco Poem

Write a poem that feels specific to San Francisco rather than generic city praise.

## Instructions

1. Match the requested form if the user specifies one: short poem, sonnet-like piece, free verse, spoken word, caption, or love letter.
2. Ground the poem in concrete San Francisco imagery. Prefer distinct details such as fog rolling over Twin Peaks, cable cars, Mission murals, the Ferry Building, Ocean Beach, the Golden Gate at dusk, steep streets, corner cafes, Victorian houses, and cold bright mornings.
3. Aim for emotional texture, not tourism brochure language. Let the city feel lived in: beautiful, weird, coastal, windy, creative, ambitious, and slightly unruly.
4. Vary sentence and line length so the poem has rhythm. Avoid repetitive praise like "beautiful city" or "amazing views" unless transformed into fresher language.
5. If no length is specified, write 12–20 lines.
6. If the user wants a stronger sense of place, read `references/sf-imagery.md` and weave in a few relevant details.
7. End on an image or line that lingers.

## Style guardrails

- Prefer fresh, sensory language over clichés.
- Do not overstuff the poem with landmarks.
- Avoid sounding like ad copy or a travel guide.
- If the user wants humor, keep it affectionate rather than snarky.
- If the user wants romance, make the city feel like a character.
